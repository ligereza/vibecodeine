# Ejemplos de mensajes rápidos para pedir trabajos a otra IA

## Ejemplo 1: crear etiqueta nueva

```txt
Lee AGENTS.md. Necesito etiquetas 10x7 cm horizontales para Impulso y Magnesio. Usa el sistema genérico. Entrega SVG editable y SVG vectorizado en ZIP. Usa colores similares a los flyers existentes. Texto base está en 01_contenido/contenido_suplementos_rd.json. Si falta información legal, deja placeholder.
```

## Ejemplo 2: cambiar medida

```txt
Lee AGENTS.md. Cambia el proyecto etiquetas_ejemplo a formato 12x8 cm horizontal, manteniendo proporción 3000x2000 px. Regenera editables, vectorizados y ZIPs.
```

## Ejemplo 3: pegar correo

```txt
Lee AGENTS.md. Te pego el correo de mi jefe. Extrae la solicitud y genera los archivos de impresión. Si falta info crítica, haz máximo 3 preguntas.

Correo:
[pegar correo]
```

## Ejemplo 4: agregar Post Fiesta

```txt
Lee AGENTS.md. Agrega ficha individual de Post Fiesta al flujo Suplementos RD con el texto de abajo. Regenera SVG editables, vectorizados, preview y ZIPs. Valida que vectorizados no tengan <text>.

Texto:
[pegar texto]
```
