## Objetivo actual

Empezar desde cero un repo limpio para construir herramientas de automatización de diseño/motion + IA.

## Herramientas a construir

2. Slowmo Blender/After Effects.
3. Asistente pedido del jefe.
4. Canva/JSON.

## Regla

No importar scripts viejos todavía. Solo usarlos como referencia manual si hace falta.

## Próximo paso

Elegir primera herramienta mínima: `flyer_eventos` o `slowmo_blender_ae`.


## Actualización integrada

Se agregó herramienta `piezas_vectoriales` para generar archivos de impresión desde JSON:

- SVG editable para Illustrator.
- SVG vectorizado con texto convertido a trazados.
- Proyectos genéricos por `config.json`.
- Proyecto real `suplementos_rd`.
- Briefs/prompts para que otra IA pueda trabajar desde un correo del jefe.

Entrada principal de la herramienta:

```txt
tools/piezas_vectoriales/SPEC.md
```
