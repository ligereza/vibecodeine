# Handoff — v0.34.0 limpieza estructural de demos/tests

Fecha: 2026-06-21
Versión destino: 0.34.0

## Objetivo

Reducir ruido del repo eliminando jobs/proyectos generados durante pruebas (`etiquetas-acme`, `uno`, `dos`, `test-pipeline`, `pieza-x`) y dejar una regla clara: los jobs reales viven localmente/ignorados; el repo solo conserva `jobs/_template` y ejemplos intencionales.

## Importante sobre este airdrop

El motor actual de airdrops copia/reemplaza archivos, pero no borra árboles. Por eso este airdrop trae un script de limpieza que debe ejecutarse desde `_airdrop` **antes** del `flujo airdrop apply`, para que el checkpoint automático incluya las eliminaciones.

## Archivos nuevos/modificados

- `scripts/cleanup_demo_artifacts.sh` — dry-run/aplicación de limpieza estructural.
- `docs/CLEANUP.md` — política de limpieza actualizada.
- `README.md`, `PARA_IA.md`, `docs/AGENT_GUIDE.md`, `pyproject.toml`, `src/flujo/version.py` — versión 0.34.0.
- Este handoff.
