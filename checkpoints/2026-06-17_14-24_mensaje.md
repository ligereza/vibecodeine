## Objetivo real del repo

Este repo sirve como **sistema de trabajo portable** para un diseñador/artista visual que:

- Trabaja en la ONG **"Reduciendo Daño"** (eventos + material visual)
- Tiene proyectos personales artísticos
- Necesita poder retomar su trabajo aunque cambie de máquina o de IA

No busca automatizar el proceso creativo, sino **reducir fricción** y mantener contexto.

## Stack técnico actual

- Paquete Python instalable (`src/flujo/`)
- CLI unificado con Typer (`flujo`)
- Descarga Instagram: **solo instaloader**
- Análisis de colores + OCR opcional
- Index SQLite
- Export a ZIP preparado para diseño
- Sistema de airdrop mejorado

---

## Reglas que se deben respetar

- **No automatizar** Photoshop, Illustrator ni Blender
- Solo instaloader para Instagram
- Mantener checkpoints y contexto para cambios de IA
- No hacer limpiezas agresivas sin consultar
- `projects/tapiz/` debe mantenerse intacto
