# flujo — arte y automatización

> Un tapiz. Un medallón. Un orden que respira.
> 
> Este repo no es una zapatilla. Es **flujo**.

`flujo` junta arte, código e IAs para que los proyectos creativos no empiecen nunca desde cero.

Cada flyer, cada etiqueta, cada rider — guarda su contexto, su input, su checkpoint. La mano sigue siendo tuya. La memoria es mía.

---

## Entrada // para IAs

Si eres una IA, lee en este orden:

```
PARA_IA.md
docs/DIMENSIONES_DEL_ORDEN.md
context/ESTADO.md
tools/flyer_eventos/SPEC.md
```

## Instalación

```
py -m pip install -r requirements.txt
```
Linux/macOS: `python3` en lugar de `py`

Dependencias ligeras: `matplotlib / pyyaml / gradio / instaloader`
**Sin yt-dlp. Sin venvs de 400 MB.**

## Herramienta activa

```
flyer_eventos
```

Descarga automática Instagram con **instaloader únicamente**.

## Comandos

```bash
# Crear flyers desde un correo con links de IG
py scripts/flyer_from_email.py "inbox/correo.txt"

# Reintentar descargas fallidas
py scripts/ig_redownload.py

# Listar / status / índice
bash scripts/flyer_list.sh
bash scripts/flyer_status_latest.sh
bash scripts/flyer_index.sh

# Pipeline inteligente
py scripts/flujo_pipeline.py "nombre pedido" inbox/correo.txt --confirm

# Interfaz web
py scripts/app.py
# o
bash start.sh

# Salud del repo
py scripts/flujo.py health
```

Flujo completo recomendado:
```
py scripts/flyer_from_email.py "inbox/correo.txt"
bash scripts/flyer_list.sh
bash scripts/flyer_index.sh
bash scripts/flyer_status_latest.sh
bash scripts/checkpoint.sh "mensaje"
```

## Piezas vectoriales

Para etiquetas / flyers de impresión / Illustrator:
```
tools/piezas_vectoriales/SPEC.md
py scripts/piezas_generar.py "projects/piezas_vectoriales/etiquetas_ejemplo/config.json"
```

## Reglas del tapiz

- Avanzar paso a paso
- No subir archivos pesados
- No automatizar Photoshop / Blender todavía
- No borrar sin confirmación
- `py` en Windows / `python3` en Linux/macOS
- Después de cada mejora: checkpoint
- **Solo instaloader. No yt-dlp.**
