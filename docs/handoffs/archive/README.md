# Archivo de handoffs

Este directorio guarda handoffs y hotfixes historicos. No es la fuente activa diaria.

Fuente activa actual:

```txt
context/LAST_HANDOFF.md
```

Subcarpetas:

- `root/`: handoffs que antes estaban en la raiz del repo y fueron archivados sin borrar historial.
- `handoffs/`: handoffs que antes estaban sueltos en el nivel raiz de `docs/handoffs/` (o en el antiguo `docs/handoffs/current/`) y fueron archivados por antiguedad (2026-06-16 a 2026-06-30).
