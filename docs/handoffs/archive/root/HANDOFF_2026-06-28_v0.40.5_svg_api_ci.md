# HANDOFF 2026-06-28 - v0.40.5 SVG API + web CI

Date: 2026-06-28
Version: 0.40.5

## Summary

SVG Visualizer now uses real repo SVGs when served by the local Python backend and keeps a demo fallback when opened without backend.

## Included

- `web/src/components/SvgVisualizer.tsx` now loads `/api/list-svg-works`.
- SVG files are fetched from `/svg/...` and rendered inline after a small safety check.
- Fallback remains `MOCK_SVG_INDEX` when backend is unavailable.
- `src/flujo/web/hub.py` accepts `/api/svg-index` as alias for `/api/list-svg-works`.
- `src/flujo/serve/server.py` now supports:
  - `GET /api/list-svg-works`
  - `GET /api/svg-index`
  - `GET /svg/...`
- `web/scripts/copy-plano.mjs` renamed to `web/scripts/copy-context.mjs`.
- Main build command is now `npm run build:context`; `npm run build:plano` remains as alias.
- CI now installs Node 20 and runs web checks before Python verify.
