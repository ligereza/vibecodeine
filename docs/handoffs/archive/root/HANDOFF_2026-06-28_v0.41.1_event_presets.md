# HANDOFF 2026-06-28 - v0.41.1 EVENTOS presets

Date: 2026-06-28
Version: 0.41.1

## Summary

Added operational presets for EVENTOS rider/plano so emails with Instagram/flyer data can be mapped quickly without asking for every operational field upfront.

## New presets

- UNDER: 2 voluntarios, 1 mesa, 2 sillas, electricidad/luz basica.
- BASE: 4 voluntarios, 2 mesas, 4 sillas, stand + testeo.
- MAINSTREAM: 8 voluntarios, 3 mesas, 8 sillas, alto flujo tipo Espacio Riesco.

## Included

- `src/flujo/eventos/presets.py`
- `docs/EVENTOS_PRESETS_RIDER.md`
- `web/src/components/PlanoTool.tsx` preset cards.
- `/api/plano/render` applies presets and returns `preset_operativo`.
- Parser adds `event_preset` for event/rider/cartelera/Instagram requests.
- `create-job-draft` stores parsed metadata in `intake.json` and `resultado.md`.
- README updated to document React web and event presets.
- Version synced to `0.41.1`.

## Example mapping

Boss email:

```txt
Hola Ligereza, te recuerdo que tenemos este evento,
https://www.instagram.com/thegrid.club/p/DYz0HOrFhJO/?hl=es

necesito que me mandes el Rider completo y subas el flyer de la cartelera.
```

Initial mapping:

```yaml
area: eventos
route: eventos_instagram_rider_cartelera
preset_operativo: mainstream
entregables:
  - rider_completo
  - flyer_cartelera
```
