## Summary

This airdrop introduces a React/Vite UI layer under `web/` while keeping flujo local/free and Python-first.

Daily use remains:

```bash
py -m flujo app
```

Node is only required to rebuild the UI.

## Included

- `web/` React/Vite/TypeScript/Tailwind source.
- `web/src/components/PlanoTool.tsx` from the Plano/Rider proposal, with a Motor Python button for `/api/plano/render`.
- `web/src/components/SvgVisualizer.tsx` and `web/src/data/svgIndex.ts` from the SVG visualizer proposal.
- `web/scripts/copy-plano.mjs` builds one single-file app and copies it to:
  - `context/plano_demo.html`
  - `context/svg_visualizer.html`
- `context/plano_demo.html` generated single-file React app.
- `context/svg_visualizer.html` generated single-file React app.
- Version sync to `0.40.4`.
- `context/LAST_HANDOFF.md` updated, ASCII-only.
- Handoff/archive/doc cleanup from earlier coherence work.
