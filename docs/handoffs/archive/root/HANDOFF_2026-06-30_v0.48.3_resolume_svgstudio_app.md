# HANDOFF 2026-06-30 - v0.48.3 Resolume + SVG Studio integrado

## Summary

Entrega integrada para la app general.

Incluye dos correcciones/avances:

1. SVG Studio: Config Editor convierte SVGs reales editables a elementos configurables.
2. Resolume/Chataigne: `py -m flujo resolume automatizar` ahora genera XML pre-flight, CSV OSC, README y `.noisette` experimental.

## SVG Studio

Problema:

- Al usar `Configurar este SVG`, los SVGs reales entraban como una sola imagen.
- Los demos eran editables porque ya estaban modelados como config JSON.

Solucion:

- Agrega parser best-effort SVG -> ConfigElement.
- Convierte elementos simples a capas editables:
  - `rect`
  - `circle`
  - `line`
  - `text`
- Si el SVG es path-heavy/vectorizado, mantiene fallback `svg_image`.
- `svg_image` queda seleccionable, movible, redimensionable y alineable.

## Resolume / Chataigne

Problema:

- Scripts externos que generan `.noisette` pueden producir JSON que Chataigne no abre si no respetan el schema real.

Solucion:

- Integra salida robusta en `src/flujo/resolume/automator.py`.
- `generate_show_automation` ahora escribe:
  - `show_automation.xml`
  - `osc_cues.csv`
  - `show_automation.experimental.noisette`
  - `README_CHATAIGNE.md`
- El XML sigue siendo la salida estable/verificable.
- `.noisette` queda marcado como experimental hasta tener una plantilla exportada desde la Chataigne real del usuario.
- UI `ResolumePanel` documenta esto.
