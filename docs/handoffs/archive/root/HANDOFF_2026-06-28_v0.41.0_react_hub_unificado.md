# HANDOFF 2026-06-28 - v0.41.0 React hub unificado

Date: 2026-06-28
Version: 0.41.0

## Summary

The daily hub is now part of the React/Vite single-file app. `context/flujo_hub.html`, `context/plano_demo.html`, and `context/svg_visualizer.html` are generated from the same React source under `web/`.

Daily use remains:

```bash
py -m flujo app
```

Node is only required to develop/rebuild the UI.

## Included

- `web/src/api/flujoApi.ts`: typed API client with fallback demo mode.
- `web/src/components/AppShell.tsx`: unified shell/navigation.
- `web/src/components/HubDashboard.tsx`: real/fallback dashboard using ping/jobs/SVG summary.
- `web/src/components/JobsPanel.tsx`: reads `/api/list-jobs`.
- `web/src/components/IntakePanel.tsx`: uses `/api/parse-real-pedido` and `/api/create-job-draft` when served by `py -m flujo app`.
- `web/src/components/CommandPanel.tsx`: copyable operational commands.
- `web/src/App.tsx`: routes by initial pathname and internal navigation.
- `web/scripts/copy-context.mjs`: copies build output to all three context HTML files.
- `src/flujo/serve/server.py`: light endpoints for `hub serve` compatibility:
  - `GET /api/list-jobs`
  - `GET /api/dashboard-summary`
  - `POST /api/parse-real-pedido`
  - read-only `POST /api/create-job-draft` response
- Version sync to `0.41.0`.
