# HANDOFF 2026-06-30 - v0.48.5 Chataigne schema real

## Summary

Hotfix para `.noisette` de Resolume/Chataigne usando el template real enviado por el usuario.

## Problema

El `.noisette` experimental anterior abria y mostraba modulos OSC/Sound Card, pero no mostraba acciones porque usaba `stateMachine` top-level.

El template real exportado desde Chataigne 1.10.3 usa:

```txt
states.items[0].processors.items[]
```

no:

```txt
stateMachine.states.items[]
```

## Solucion

- Reescribe `build_chataigne_noisette_experimental` para seguir el schema real 1.10.3.
- Usa top-level `states`.
- Usa `commandModule: "osc"` en minuscula, como el template real.
- Usa `/modules/soundCard/values/ltc/ltcTime`.
- Usa rutas `consequencesTRUE` en el argumento OSC, como Chataigne serializa.
- Mantiene layout realista copiado del template.
- Mantiene XML y CSV como salidas estables.
