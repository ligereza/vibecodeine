# HANDOFF — v0.34.9 Sincronización de documentación para agentes

Fecha: 2026-06-22

## Resumen

Airdrop pequeño de documentación. No cambia lógica de ejecución; sincroniza las
guías que los agentes deben leer antes de trabajar.

## Por qué

Tras aplicar v0.34.8, el repo funciona y las pruebas pasan, pero varias guías
obligatorias seguían diciendo versiones antiguas:

- `PARA_IA_CONTEXT.md`: v0.25.0
- `docs/AGENT_GUIDE.md`: v0.34.7
- `docs/CLI.md`: v0.16

Eso contradice el protocolo del README, que exige mantener la documentación
relevante alineada con cada airdrop. Este drop corrige esa deuda.
