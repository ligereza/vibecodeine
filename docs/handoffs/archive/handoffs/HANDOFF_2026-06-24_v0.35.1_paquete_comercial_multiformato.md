# HANDOFF — v0.35.1 Paquete comercial multiformato

Fecha: 2026-06-24

## Resumen

Avance solicitado para el pendiente real:

> Necesito enviar un brief que explique la estructura imagen / texto de flyers, etiquetas, pendones, post de instagram junto con una cotización.

## Cambios

- Nuevo módulo `src/flujo/comercial/multiformato.py`.
- Nuevo comando:

```bash
py -m flujo brief paquete-cotizacion <job_o_txt>
```

- Genera en `salida_comercial/`:
  - `brief_estructura_multiformato.md`
  - `tabla_formatos.md`
  - `cotizacion_base.md`
  - `preguntas_para_cerrar.md`
  - `manifest.json`
- El parser de intake ahora detecta este tipo como:
  - `tipo: paquete_cotizacion`
  - `formato: paquete_comercial_multiformato`
  - `tool: brief paquete-cotizacion`
- `brief_from_text` ya no colapsa estos pedidos a `etiqueta` por mencionar etiquetas dentro de la lista.
- Tests nuevos en `tests/test_comercial_multiformato.py`.

## Uso recomendado

```bash
mkdir -p inbox
cat > inbox/brief_multiformato_cotizacion.txt <<'EOF'
Necesito enviar un brief que explique la estructura imagen / texto de:
- flyers
- etiquetas
- pendones
- post de instagram
junto con una cotización.
EOF

py -m flujo job new "brief estructura imagen texto cotizacion multiformato" --email inbox/brief_multiformato_cotizacion.txt
py -m flujo job prepare jobs/<job>
py -m flujo brief paquete-cotizacion jobs/<job> --cliente "Nombre Cliente"
```

Luego revisar `jobs/<job>/salida_comercial/cotizacion_base.md` y completar precios antes de enviar.
