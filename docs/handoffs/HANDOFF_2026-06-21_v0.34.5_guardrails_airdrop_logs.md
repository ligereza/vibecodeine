# Handoff — v0.34.5 guardrails de airdrop y logs

Fecha: 2026-06-21
Versión destino: 0.34.5

## Objetivo

Evitar que se repitan errores de airdrops defectuosos: ZIPs vacíos, archivos generados, rutas deformadas por Markdown, checkpoints con pruebas fallidas y logs difíciles de compartir.

## Cambios

- `scripts/validate_airdrop.py`: valida `_airdrop/` antes de aplicar.
- `scripts/run_airdrop_checks.py`: aplica airdrop, ejecuta pruebas y solo hace checkpoint si todo pasa.
- `.gitignore`: ignora `_logs/` y logs locales de airdrop.
- `docs/AIRDROP_REVIEW.md`: flujo operativo del dueño.
- `docs/AGENT_AIRDROP_PROTOCOL.md`: protocolo para agentes externos.
- `README.md`, `PARA_IA.md`, `docs/AGENT_GUIDE.md`: referencias al nuevo flujo.
- `src/flujo/version.py`, `pyproject.toml`: versión 0.34.5.
