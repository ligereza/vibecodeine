# HOTFIX — v0.34.11 Seguridad hub, export ZIP e intake

Fecha: 2026-06-23

## Resumen

Airdrop de correcciones para bugs detectados por auditoría:

- Bloqueo de path traversal en el servidor local del hub.
- Servidor del hub migrado a `ThreadingHTTPServer` para que `/api/events` no congele otras APIs.
- `export_flyer()` ahora incluye `compose.jsx`, `compose_ai.jsx` y `blender_setup.py` dentro del ZIP final.
- Parser de pedidos detecta `plano`, `stand`, `cotización` y `cartelera` antes de caer en `flyer`.
- `flyer-import` soporta URLs Instagram con usuario (`/user/p/CODE/`), sin esquema y `/reels/`.
- Índice SQLite inicializa esquema en consultas fresh (`list_flyers`, `find_duplicates`).
- Setup dev instala `pre-commit`.
- Generador SVG preserva llaves literales en textos humanos (`Precio {especial}`) en vez de crashear.
- Dashboard HTML escapa campos dinámicos.
- Recepción IMAP de airdrops usa `sys.executable`, extracción ZIP segura y opt-in para cambios al motor.
- Jobs respetan `FLUJO_WORKSPACE_ROOT` en dev/tests para no escribir en el repo por accidente.
