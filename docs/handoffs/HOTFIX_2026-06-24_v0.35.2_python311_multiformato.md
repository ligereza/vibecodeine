# HOTFIX — v0.35.2 Python 3.11 para paquete comercial multiformato

Fecha: 2026-06-24

## Motivo

Al aplicar v0.35.1 en Windows/Python 3.11, `compileall` falló con:

```text
SyntaxError: f-string expression part cannot include a backslash
```

en `src/flujo/comercial/multiformato.py`.

El código pasaba en Python 3.12+ por PEP 701, pero el proyecto corre CI/Windows con Python 3.11.

## Cambio

Se precalculan variables markdown antes del f-string:

- `formats_bullets`
- `sections_md`
- `source_clean`

Así el f-string ya no contiene expresiones con `\n` dentro.

## Instrucción

Aplicar encima de v0.35.1 fallido y reanudar:

```bash
py scripts/run_airdrop_checks.py --resume "v0.35.2 - hotfix python311 paquete comercial multiformato"
```

También se puede aplicar normal si se limpió `_airdrop/` y se usa el ZIP nuevo.
