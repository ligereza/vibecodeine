# Checklist de impresión / entrega gráfica

Usar antes de entregar SVG/PDF/ZIP.

## Brief

- [ ] Tipo de pieza confirmado: etiqueta / flyer / sticker / tarjeta / otro.
- [ ] Medida real confirmada.
- [ ] Orientación confirmada.
- [ ] Cantidad de piezas confirmada.
- [ ] Producto(s) confirmado(s).
- [ ] Fecha límite registrada.

## Técnica

- [ ] Sangrado confirmado o marcado como pendiente.
- [ ] Área segura considerada.
- [ ] Formato final requerido confirmado: SVG / PDF / AI / PNG / ZIP.
- [ ] Versión editable generada.
- [ ] Versión vectorizada generada.
- [ ] Vectorizados revisados sin `<text>`.

## Contenido

- [ ] Texto final aprobado o marcado como pendiente.
- [ ] Claims legales/sanitarios no inventados.
- [ ] Ingredientes/nutrientes revisados.
- [ ] Modo de uso revisado si aplica.
- [ ] Advertencias legales revisadas si aplica.
- [ ] QR/código/lote/vencimiento confirmado o placeholder visible.

## Entrega

- [ ] Preview revisado.
- [ ] ZIP editable creado.
- [ ] ZIP vectorizado creado.
- [ ] ZIP completo creado si aplica.
- [ ] Ruta de entrega anotada en `resultado.md`.
