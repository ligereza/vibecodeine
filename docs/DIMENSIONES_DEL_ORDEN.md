# Dimensiones del Orden

Este repo funciona como una zapatilla/alargador organizador para proyectos creativos, automatizaciones e IAs.

No reemplaza el trabajo manual del diseñador. Ordena entradas, salidas, contexto y pasos para trabajar más rápido.
