# Quality gates

Antes de cerrar un trabajo, pasar por estas puertas:

## Gate 1 — Entrada

- Job creado.
- Pedido original guardado.
- Privacidad revisada.
- Brief extraído o escrito.

## Gate 2 — Diseño/proyecto

- Config JSON válido.
- Formato/proporción confirmada.
- Componentes requeridos insertados.
- Pendientes críticos resueltos o documentados.

## Gate 3 — Render

- SVG editable generado.
- SVG vectorizado generado.
- Preview generado.
- ZIPs generados.

## Gate 4 — Entrega

- `flujo_health` OK.
- `piezas_check_outputs` OK.
- Manifiesto de entrega creado.
- Respuesta al usuario preparada.
