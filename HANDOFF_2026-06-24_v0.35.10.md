# HANDOFF - v0.35.10 Gmail subject routing + agent README

ASCII-only file. Owner uses Windows + Git Bash. Use `py`, not `python`.

## What changed

User clarified that email subjects should not need the word "flujo".
Routes now use normal subjects:

- subject contains `eventos` or `evento` -> EVENTOS
- subject contains `suplementos` or `suplemento` -> SUPLEMENTOS

Recommended Apps Script property:

```txt
GMAIL_ROUTES = {subject:eventos subject:evento}|EVENTOS|pedido,area/eventos,estado/por-revisar,gmail,instagram,action/descargar-ig;{subject:suplementos subject:suplemento}|SUPLEMENTOS|pedido,area/suplementos,estado/por-revisar,gmail
```

Only the processed label is needed:

```txt
GMAIL_LABEL_DONE = flujo-procesado
```

Legacy label routes still work if needed:

```txt
label:flujo-eventos|EVENTOS|...
```
