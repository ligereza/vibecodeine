📖 Creating Plugins

Each plugin lives in plugins/<name>/ and must contain:

  • __init__.py — exports plugin_class
  • manifest.json — name, version, description, permissions
  • Optional: static/, config.json, templates/

The plugin class inherits from PluginBase and gets access to:

  • self.controller — XiaomiController instance
  • self.register_route() — register Flask routes
  • self.get_config() / self.set_config() — persistent config
  • self.data_dir — per-plugin data folder
  • self.context.schedule() — background tasks
