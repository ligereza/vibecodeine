# La Gota - experiencia web de testeo (concepto RD)

La dinamica fundacional de RD: la gotita de reactivo cayendo en la sustancia,
como experiencia web ABSTRACTA (no figurativa). Canvas generativo: la gota
cae, impacta el fluido y el color de la reaccion florece en ondas y manchas
organicas.

Version 2 (mejorada): gotero abstracto que forma y suelta la gota, superficie
de agua con fisica de resortes acoplados, salpicadura + ondas, tinta que
difunde con campo de flujo (blending aditivo), EVOLUCION TEMPORAL del color
(evo:[inicial,final] — el timing es parte de la lectura real), sonido de
goteo sintetizado (WebAudio, sin assets), modo instalacion/proyeccion para
el stand, y EL LIENZO DE LA NOCHE: cada gota mancha permanentemente el
canvas colectivo, descargable como PNG (obra generativa del evento).

Funciones:
- Seleccion de reactivo (Marquis/Mecke/Simon's - TABLA DEMO) y color
  observado por swatches, o MUESTREO POR CAMARA (getUserMedia: tocas el
  color real de tu reaccion y hace match perceptual al mas cercano).
- Informa la reaccion asociada + aviso permanente: el testeo colorimetrico
  es orientativo, nunca confirmatorio, nada avala consumo.
- Base de datos anonima: localStorage + exportar JSON + constante ENDPOINT
  para webhook/Apps Script cuando RD quiera centralizar (dato agregado y
  diferido segun el modelo de seguridad del Semaforo RD).
