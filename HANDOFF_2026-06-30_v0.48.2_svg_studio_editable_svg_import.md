# HANDOFF 2026-06-30 - v0.48.2 SVG Studio editable SVG import

## Summary

Hotfix para SVG Studio / Config Editor.

Problema real:
- "Configurar este SVG" cargaba SVGs reales como una sola imagen/capa `svg_image`.
- Los demos (Etiqueta RD y Flyer minimo) si eran editables porque ya eran `config.json` nativos.
- El usuario necesita que SVGs reales editables se puedan configurar, no solo mover como imagen.

Solucion:
- Agrega un conversor best-effort SVG -> ConfigElement.
- Convierte elementos simples del SVG real a capas editables del Config Editor:
  - `rect`
  - `circle`
  - `line`
  - `text`
- Si el SVG es muy complejo/vectorizado y no hay suficientes elementos simples, mantiene fallback `svg_image`.
- `svg_image` tambien queda mejorado: se renderiza dentro del SVG editor y tiene propiedades X/Y/ancho/alto/opacidad.
- El overlay HTML absoluto anterior fue eliminado.
- `flujo hub serve` sube limite de SVGs por grupo de 8 a 50 para alinearse con el hub principal.

## Importante

No se borraron SVGs reales. En el clon limpio solo se encontraron SVGs de suplementos/base actualmente usados. Borrar outputs reales requiere una auditoria de formatos separada para no perder trabajo.
