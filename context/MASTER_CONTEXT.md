# MASTER CONTEXT — Flujo IA + Diseño

Última actualización: PENDIENTE

## Objetivo del sistema

Crear un flujo gratuito para trabajar como diseñador gráfico usando varias IAs gratuitas, Arena, VS Code, GitHub y automatizaciones locales, sin tener que subir todos los archivos cada vez.

## Cómo debe ayudar una IA nueva

Cuando una IA reciba este repo o este contexto, debe:

1. Leer este archivo primero.
2. Leer el checkpoint más reciente en `/checkpoints`.
3. Revisar `/logs/CHANGELOG.md`.
4. Proponer el siguiente paso más pequeño y accionable.
5. No asumir que todos los archivos pesados están disponibles.
6. Trabajar con previews, resúmenes, prompts y notas.

## Convenciones

- Un checkpoint debe explicar qué se intentó, qué funcionó, qué quedó a medias y qué debería hacer la próxima IA.
- Los archivos pesados no se suben salvo que sea necesario.
- Los prompts importantes se guardan en `/prompts`.
- Las respuestas importantes de IA se guardan dentro de cada proyecto o en `/logs`.
