Datadrop — Airdrop inverso (fotos de trabajos TERMINADOS)
Sube fotos de piezas ya entregadas para construir una base visual útil en el hub. El proceso es simple: arrastra una imagen, asigna tipo y guarda el datadrop.
Todo se procesa localmente para mantener la privacidad y dejar una referencia real para revisión y futura mejora.
Foto de pieza terminada
