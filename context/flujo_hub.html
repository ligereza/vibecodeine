🚀 Export directo a tu flujo (Illustrator / Photoshop / Blender)

Flujo recomendado:
Pedido (texto) → Hub o CLI → flujo render → Export → Abre en AI / PS / Blender
Adobe Illustrator (vector / impresión)
  1. flujo render run projects/.../config.json
  2. Abre el SVG en AI (editable + vectorizado)
  3. Usa swatches desde aistetic.json
  4. Opcional: corre ai/compose_ai.jsx desde la carpeta
Ideal para piezas vectoriales, flyers, etiquetas, riders.
Adobe Photoshop (digital / foto)
  1. flujo export proyecto/ (genera ZIP)
  2. Descomprime y abre carpeta working/
  3. Corre compose.jsx (File → Scripts → Browse)
  4. Coloca fotos + aplica paleta aistetic
Perfecto para carteleras, posts IG, digital.
Blender (3D)
  1. Prepara con flujo plano ... o config
  2. En Blender: Text Editor → Open → blender_setup.py
  3. Ejecuta: setup_stand("ruta/al/json")
  4. Materiales + dimensiones desde aistetic
Para stands, carteleras 3D, recursos (frascos).
Tip: Después de render/export los archivos ya tienen estilo aistetic. Usa el hub para pegar el pedido y copiar el comando correcto.
