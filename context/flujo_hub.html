Cómo delegar desde el hub (guía práctica para humano y agentes IA)
1. Ejecuta `flujo app` (o `flujo app --desktop`) — entrada diaria obligatoria única. El hub + APIs reales + live se activan (o usa fallback estático).

3. Selecciona 1+ roles (checkboxes = delegación paralela real a los 5: Visual Polish / Pipeline / Brand / Future / Packaging).

5. Lanza en clones separados (git clone / terminal independiente). Cada sub-agente: `flujo app` + lee LAST_HANDOFF + sigue su prompt.
6. Cada uno entrega airdrop propio. Principal integra + actualiza LAST_HANDOFF central.
