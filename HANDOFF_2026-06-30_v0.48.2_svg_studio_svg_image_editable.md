# HANDOFF 2026-06-30 - v0.48.2 SVG Studio svg_image editable

## Summary

Hotfix para SVG Studio / Config Editor.

Problema detectado:
- Al usar "Configurar este SVG", el SVG importado entraba como `svg_image`.
- Pero se renderizaba como overlay HTML fuera del SVG editor.
- Resultado: capa fantasma, no seleccionable/editable de forma normal, dificil de mover/resize/alinear.

Solucion:
- `svg_image` ahora se renderiza dentro del SVG editor con `<image href="data:image/svg+xml...">`.
- Se puede seleccionar, mover, redimensionar y alinear como una capa normal.
- PropEditor agrega X/Y/ancho/alto/opacidad para `svg_image`.
- Se elimina el overlay HTML duplicado.
- `flujo hub serve` sube limite de SVGs por grupo de 8 a 50 para alinearse con hub principal.
