# HANDOFF v0.43.0 - El Cerebro Operativo de la ONG

## Cambios Clave
- **Rider Pro:** Integración total de Requerimientos + Plano en un documento imprimible profesional.
- **Iconografía Técnica:** Símbolos profesionales (Power, Heat, Rack, Extinguisher, Water) sin emojis.
- **Configuración:** Paneles de ajustes persistentes para textos de la ONG y lógica de precios.
- **Cotización Operativa:** Cálculo automático por Presets y soporte para Items Manuales.
- **Flyers v4.1:** Rediseño Techno-Elegant 2800x2000px.

## Instrucciones
1. Corre `npm run build:context` en web/ para activar la nueva UI.
2. Usa el botón "Imprimir Rider" para ver el formato de documento de ingeniería.
